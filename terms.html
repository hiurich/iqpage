2. Description of Service

IQPage is an AI-powered Chrome browser extension that provides:

  • Automatic summarization of web page content
  • Question and answer chat about page content
  • Smart text highlighting with explanation, translation, and note-taking features
  • Niche-specific analysis (legal, academic, social, consulting, science)
  • Bias detection and article comparison (paid plans)
  • Export of summaries to PDF and Word format (paid plans)

The Extension uses artificial intelligence provided by Anthropic's Claude API to process content.
